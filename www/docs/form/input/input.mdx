## Base

Input может быть как контролируемым, так и нет. В случае неконтролируемого варианта необходимо передать проп uncontrolled. Для контролируемого компонента, разработчику необходимо предоставить состояние и его обработчик для инпута через проперти <b>value</b> и <b>onChange</b>.

Для контролируемого варианта c каунтером и лимитом вводимых символов пакет дополнительно экспортирует хук <i>useLimitCounter</i>, содержащий в себе необходимые пропсы.

<p>
  <i>{`const { value, onChage, onClearInput, endMetaText } = useLimitCounter(42);`}</i>
</p>
